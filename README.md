## Feature
* Shopping cart
* Search products
* Order detail
* JWT authentication & authorization 
* [Stripe checkout](https://stripe.com/docs/payments/checkout)
* Responsive


## Tool
* React
* Typescript
* RTK(Redux Tool Kit)
* Express
* Mongoose
